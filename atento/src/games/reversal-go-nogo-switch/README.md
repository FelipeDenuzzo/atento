# Reversal Go/No-Go Switch

Treino de atenção alternada e flexibilidade cognitiva com reversão de mapeamento estímulo-resposta.

## Regras

- **NORMAL**: clique apenas na estrela.
- **INVERTIDO**: clique apenas quando **não** for estrela.

Cada trial indica o cue da regra, depois mostra o estímulo.

## Saída

O jogo gera exportação em:

- `.txt`
- `.json`
- `.csv`

Com métricas de acurácia, RT, erros de comissão/omissão e custo de troca (`switch cost`).
